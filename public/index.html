What is JQuery

JQuery is a js library; It's still javascript, we're just utilizing some new functions!

More Clearly:

This is one way to define a function: function funcName(args){things this function does}

This is how we call that function: funcName()

Here's another function: JQuery()

That JQuery function has been given another name to make typing faster (the dollar sign): $()

So the JQuery function is a function that returns a JQuery object to the thing that is passed as its argument
